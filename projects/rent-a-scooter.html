Rent a Scooter

Scooter rental prototype where users can browse, book and earn points.

Project overview

Challenge

The challenge was to dive into InVision to gain an understanding of its prototyping capabilities. I aimed to create a dynamic and interactive prototype that not only demonstrated functionality but also allowed me to refine my skills in using this tool.


The idea of this project came from a similar app, but for car rentals. Since the focus was more on mastering the program itself rather than designing the user experience, I won't be adding a detailed design process for this project.

My objectives

  • Gain a thorough understanding of how to use InVision's interactive features to create dynamic prototypes.
  • Build a prototype that effectively demonstrates the user journey, from browsing to booking, with smooth transitions and interactions.

Design and prototype

Style

To help improve visual ergonomics by reducing eye strain, I decided to use a dark theme for the application. For the theme colors, I selected less saturated colors that don't visually vibrate against a dark background.

High-fidelity

For this project, I designed a series of high-fidelity screens. These screens were created with the goal of bringing the user journey to life in a prototype, demonstrating the key steps involved in renting a scooter. While my main focus was on testing the prototype capabilities, I still made an effort to ensure the design was visually appealing.
